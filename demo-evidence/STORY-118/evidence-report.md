## Input Fixture

**File:** `empty_ua_flood.pcap` (generated by `make_empty_ua_pcap.py`)

- 25 TCP flows, 125 packets total
- Each flow: SYN → SYN-ACK → ACK → HTTP GET (PSH+ACK) → FIN+ACK
- Each HTTP request: `GET /reqNNN HTTP/1.1\r\nHost: example.com\r\nUser-Agent:\r\nConnection: close\r\n\r\n`
- `Host:` is always present/non-empty → avoids missing-Host co-finding
- `User-Agent:` is present-but-empty (`Some("")`) → fires empty-UA finding per `src/analyzer/http.rs:359`
- Distinct URIs (`/req001`–`/req025`) → each finding has distinct evidence string
